# Blood Moon for Bright Nights

Adds an event that happens at configurable intervals (current default is 7 days) which causes nearby hordes to be attracted to your location, move much faster, and spawns additional hordes nearby to supplement population loss.
The goal is to improve defense gameplay and make static bases more engaging.
